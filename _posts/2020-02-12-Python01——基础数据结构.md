# Print格式化
```python3
print('Hello, %s' % 'world')
#Hello, world
print('Hi, %s, you have $%d.' % ('Michael', 1000000))
#Hi, Michael, you have $1000000.
print('%2d-%02d' % (3, 1))
#3-01
print('%.2f' % 3.1415926)
#3.14
#用%%来表示一个%
print('Hello, {0:05d},  {1:.1f}%'.format(9527, 17.125))
#Hello, 09527,  17.1%
```
# 数据类型
- ## 整数
- ## 浮点数
- ## 字符串
  - 转义字符`\`
  - 多行字符串`''' '''`或`r''' '''`
- ## 布尔值
  - `True / False`
  - 布尔运算`and / or / not`
- ## 空值
  - `None`
- ## 列表
  - Python内置的一种数据类型是列表
  - 一种有序的集合，可以随时添加和删除其中的元素
  - `lst = ['Michael', 'Bob', 'Tracy']`
  - `len(lst)`返回列表元素个数
  - 索引号从`0 ~ len(lst)-1`, `-1`表示从尾部开始
  - `lst.append(var)`，向列表尾部添加变量
  - `lst.insert(1,var)`，将变量添加到索引为1的位置
  - `lst.pop(i)`，删除指定索引的元素，不带参则删除尾部元素
  - `lst.clear()`，清空列表
  - `lst.copy()`，返回一个深拷贝的列表
  - `lst.count(i)`，返回`i`出现的次数
  - `lst.reverse()`，列表反序
  - `lst.index(i)`，返回首次出现`i`的位置
- ## 元组
  - 另一种有序列表叫元组
  - 但是元组一旦初始化就不能修改
  - 在定义的时候，元组的元素就必须被确定下来
  - `t=()` or `t=(1,)` or `t=(1,2)`
  - `tup = ('Michael', 'Bob', 'Tracy')`
  - `tup.count(i)`，返回`i`出现的次数
  - `tup.index(i)`，返回首次出现`i`的位置
- ## 字典
  - Python内置了字典
  - 在其他语言中也称为map，使用键-值（key-value）存储
  - 具有极快的查找速度
  - `d = {'Michael': 95, 'Bob': 75, 'Tracy': 85}`
  - `d['Adam'] = 67`，不存在则新增，存在则更新
  - `'Thomas' in d`，判断是否存在
  - `d.get('Thomas')`，不存在则返回`None`
  - `d.get('Thomas',-1)`，不存在则返回`-1`
  - `d.pop('Bob')`，删除对应键，返回对应值
  - `d.copy()`，返回一个深拷贝的字典
  - `d.clear()`，清空字典
  - `d.keys()`，返回键
  - `d.values()`，返回值
  - `d.items()`，返回键值对
- ## set
  - set和dict类似，也是一组key的集合，但不存储value
  - 由于key不能重复，所以，在set中，没有重复的key
  - 可以看成数学意义上的无序和无重复元素的集合`&交集 |并集`
  - 常用于数组（列表）去重
  - `s=([1,2,3,1])`,`s={1,2,3}`
  - `s.add(4)`，添加元素
  - `s.remove(key)`，删除元素

# 函数
- ## 内置函数
  - `abs(var)`，返回绝对值
  - `max(a,b,c,...)`，返回最大值
  - `min(a,b,c,...)`，返回最小值
  - `int()`，把其他数据类型转换为整数
  - `float()`，把其他数据类型转换为浮点数
  - `str()`，把其他数据类型转换为字符串
  - `bool()`，把其他数据类型转换为布尔值
  - `list()`，把其他数据类型转换为列表
- ## 自定义函数
  - 默认参数`def fn(x=0):`
  - 可变参数`def fn(*arr):`，可传递任意个数的参数（元组）
  - 关键字参数`def fn(**dic):`，可传递整个字典
  - 限制关键字`def person(name, age, *, city, job):`，限制参数为` * `后面的`city , job`
```python3
# test.py
def myfn(x):
  if x==0:
    return False
  else:
    return True

  pass

def add_end(L=None):
    if L is None:
        L = []
    L.append('END')
    return L
```

- ## 列表生成式
```python3
list(range(1, 5))===[x for x in range(1,5)]===[1, 2, 3, 4]
[x * x for x in range(1, 11)]===[1, 4, 9, 16, 25, 36, 49, 64, 81, 100]
[x * x for x in range(1, 11) if x % 2 == 0]===[4, 16, 36, 64, 100]
[m + n for m in 'AB' for n in 'XY']===['AX', 'AY', 'BX', 'BY']

d = {'x': 'A', 'y': 'B', 'z': 'C' }
[k + '=' + v for k, v in d.items()]===['y=B', 'x=A', 'z=C']

L = ['Hello', 'World', 'IBM', 'Apple']
[s.lower() for s in L]===['hello', 'world', 'ibm', 'apple']
```
